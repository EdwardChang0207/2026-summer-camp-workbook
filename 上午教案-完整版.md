# 上午授課｜完整教案（四天）

> **教學平台**：頑扣官網 Python 魔法夏令營「**學院**」模式
> 入口：https://www.letsfunnycoding.com/python-summer-camp?mode=lobby → 進入大廳後點 **🏛️ 學院** → 選當日 Day。
> **教學形態**：老師**螢幕分享**官網「學院」章節，邊講邊示範；學生各自在自己瀏覽器開同一章節跟著操作、打小題，即時批改。
> **每章節結構**：劇情情境 → 觀念說明（常配概念動畫）→ 3～5 道小題逐題解鎖。**一次一觀念、不洩答**。
> **線上跟做方式**：學生有問題在視訊聊天室輸入「？」或舉手示意；老師暫停分享、直接在視訊中引導，不直接給答案。

---

## 共同教學原則（頑扣鐵則）

1. **一次一觀念**：一個章節只推進一個新概念，講完立刻讓學生動手。
2. **觀念配動畫/比喻**：抽象概念先用生活比喻或官網動畫建立直覺，再寫程式。
3. **不洩答**：學生卡關用「提問」引導，不直接給答案。
4. **免捲動原則**：螢幕分享時一個觀念儘量一個畫面講完，學生不必上下捲找重點。
5. **跑了再往下**：每寫一段就執行看結果，確認對了再加下一段。

# 📗 Day 2：判斷！if / else 🤔
**副標**：比較、條件、邏輯，讓程式會做選擇　｜　**主題色**：綠　｜　**地下城樓層**：B2F 迷霧之門
**下午專案銜接**：職業選擇與劇情分支（if/else 判斷職業）

### 本日學習目標
- 會用比較運算子 `== != > < >= <=` 產生 True / False
- 會用 `if`、`if/else`、`if/elif/else` 做單一與多重判斷
- 理解縮排決定「程式屬於哪個區塊」
- 會用 `and / or / not` 組合多個條件

### 章節順序與教學重點

| 章節 | 觀念 | 教學要點 / 比喻 | 易錯點 |
|------|------|----------------|--------|
| **2-1 比較運算子** | `== != > < >= <=` | 比較的結果是「是非題」True/False。`==` 是「相等嗎？」 | 把 `=`（賦值）誤用成 `==`（比較） |
| **2-2 if 第一次判斷** | `if`、縮排 | 「如果……就……」。`if` 後面冒號 `:`，下一行**縮排**才算 if 的內容。 | 忘冒號、縮排不一致 → IndentationError |
| **2-3 if / else 二選一** | `if/else` | 「成立走這邊，不成立走那邊」，兩條路只會走一條。 | else 後面誤加條件 |
| **2-4 elif 多重判斷** | `if/elif/else` 門檻 | 像成績分級：先判最高門檻，依序往下。順序錯結果就錯。 | elif 順序排錯（先寫低門檻） |
| **2-5 and / or 邏輯** | `and / or / not` | `and`＝兩個都要成立；`or`＝有一個成立就好；`not`＝反過來。 | 把「且/或」用反；條件沒加完整 |

# 📙 Day 3：重複！迴圈 🔁
**副標**：`for`、`while`、`break`、`continue`，讓程式自動重複　｜　**主題色**：橙　｜　**地下城樓層**：B3F 骸骨墓穴
**下午專案銜接**：回合制戰鬥系統（while 迴圈）

### 本日學習目標
- 會用 `for` + `range()` 跑固定次數
- 會用累加器（`+=`）做總和統計
- 會用 `while` 依條件重複，並設定正確的終止條件
- 會用 `break`（提前跳出）與 `continue`（跳過本輪）

### 章節順序與教學重點

| 章節 | 觀念 | 教學要點 / 比喻 | 易錯點 |
|------|------|----------------|--------|
| **3-1 for + range()** | `for`、`range()`、步長 | 「重複做 N 次」。`range(3)` 是 0,1,2。可設起點、終點、步長。 | range 邊界 off-by-one |
| **3-2 累加計算：總和** | 累加器、`+=` | 準備一個「存錢筒」變數，每輪往裡加。`total += x` 是 `total = total + x` 的簡寫。 | 累加器忘了在迴圈外先歸零 |
| **3-3 while 條件式重複** | `while`、終止條件 | 「只要條件成立就一直做」。**迴圈裡要有東西讓條件愈來愈接近結束。** | 無限迴圈（條件永遠成立） |
| **3-4 break 提前跳出** | `break`、`%` 餘數 | 「達成目標就立刻離開」。`%` 取餘數可判斷整除/奇偶。 | break 位置放錯層 |
| **3-5 continue 跳過某次** | `continue` | 「這一輪不做了，直接下一輪」。常用來過濾不要的情況。 | continue 後仍誤以為會執行下面 |
| **3-6 回合制戰鬥（整合）** | `while`+`-=`+計數器+f-string | 把上面全部串起來：HP 累減、回合計數、條件結束。**直接是下午專案的核心。** | 漏掉讓 HP 變化 → 打不完 |
